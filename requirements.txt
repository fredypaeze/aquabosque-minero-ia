# Dependencias del proyecto AquaBosque Minero IA

# Manejo de datos
pandas
numpy
openpyxl  # necesario para que pandas lea archivos .xlsx (DIVIPOLA), Fase 3A

# Geoespacial (liviano)
shapely  # validar geometrias (nulas/tipo/validez) del catastro minero ANM, Fase 3C.
         # Se elige shapely en vez de geopandas: solo se necesita construir
         # geometrias desde GeoJSON y chequear is_valid/tipo; geopandas exige
         # ademas fiona/pyproj/GDAL (mucho mas pesado de instalar) para
         # funcionalidad de I/O que aqui no hace falta, porque el GeoJSON ya
         # se lee/escribe con json estandar.
pyproj   # transformar/validar coordenadas entre CRS (EPSG:4326 <-> EPSG:9377
         # MAGNA-SIRGAS 2018 / Origen-Nacional) para los limites municipales
         # DANE, Fase 3D. Necesario porque shapely no incluye reproyeccion;
         # pyproj es mucho mas liviano que agregar geopandas solo para esto.
rasterio # leer/inspeccionar/reproyectar rasters (bosque/cambio de bosque IDEAM,
         # formato GeoTIFF via WCS), Fase 2D.1. Trae su propio GDAL empaquetado
         # en el wheel de Windows, no requiere instalacion de GDAL aparte.

python-dotenv
pyyaml
