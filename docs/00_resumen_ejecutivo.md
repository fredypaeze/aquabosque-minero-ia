# Resumen Ejecutivo — AquaBosque Minero IA

**AquaBosque Minero IA** es un sistema explicable de IA geoespacial que integra **7 fuentes de datos abiertos oficiales** (ANM, IDEAM, RUNAP, DANE) para **priorizar** los municipios colombianos donde confluyen señales de presión minera, deforestación y afectación hídrica.

**No prueba causalidad, no acusa, no sanciona.** Prioriza territorios para orientar revisión técnica y monitoreo institucional basado en evidencia.

## Resultados
- **1,122 municipios** analizados con las 5 dimensiones de riesgo.
- Priorización: **57 Crítico · 112 Alto · 281 Medio · 672 Bajo**.
- Territorios top: ORITO, SAN JOSÉ DEL GUAVIARE, SAN VICENTE DEL CAGUÁN, LA MACARENA, PUERTO GUZMÁN.
- Modelo XGBoost + SHAP explicable; accuracy 91.1% (línea base 59.8%).

## Valor público
Convierte datos sectoriales dispersos en **priorización ambiental accionable**, reproducible y defendible, útil para el Ministerio de Minas y Energía y comprensible para decisión pública.
